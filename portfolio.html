Key Projects

Skill Swap — Peer Learning Platform

Designed a digital barter system that allows users to exchange skills using AI-based matching. Created BMC, prototype, and marketing plan integrating user analytics for retention optimization (Nielsen Norman Group, 2023).

Case Summary → SkillSwap is an innovative mobile platform designed to democratize learning through skill exchange without money. Built on the principle that everyone has valuable knowledge to share, it connects students, professionals, and retirees via a community-driven model.

Eco Rewards — Recycling Incentive App

Developed a gamified recycling concept defining key metrics such as Monthly Active Users (MAU) and redemption rates to measure behavioral impact. Applied W3C accessibility and sustainable UX frameworks (W3C WAI, 2023).

Case Summary → Eco Rewards is a smartphone-based incentive system rewarding users for responsible waste disposal. Users scan a QR code on smart bins and earn redeemable points for discounts at local shops and online platforms.

Netflix PMO Optimization Proposal

Applied PESTEL and Porter’s Five Forces frameworks to design a balanced-authority PMO aligning innovation with governance. Proposed KPIs and success metrics supported by AI-enabled dashboards (Quarto, 2024).

Sharma Traders — E-Commerce Jewelry Brand

Founded and launched an Indian-style jewelry e-commerce brand on Shopify. Implemented digital marketing, custom packaging, and automated order management achieving 300+ monthly visitors in 3 months.
